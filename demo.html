Python 程式整理

Python 的官方網站

Variables 變數

變數可以儲存內容

a = 5
b = 3
c = a + b

Strings 字串

a = "Hello World"
len(a) # 長度
a[0] # 第一個字

Lists 列表

a = [1, 2, 3, 4, 5]
len(a) # 長度
a[0] # 第一個字

Dictionaries 字典

fruits = {
"apple": "蘋果",
"banana": "香蕉",
"orange": "橘子"
}
fruits["apple"] # 蘋果

Booleans 布林

a = True
b = False
a and b is False
a or b is True
not a is False

if... else... 判斷式

pass = "12345abc"
if pass == "12345abc":
    print("登入成功")
else:
    print("登入失敗")

For Loops For迴圈

a = [1, 2, 3, 4, 5]
for i in a:
    print(i)

While Loops While迴圈

while input("Enter your name: ") != "Neil":
    print("Not me! ")

Functions 函數

def f(x):
    return x * 20 + 3
